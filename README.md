# auto-reduce-aigc

自动降低aigc率

一个 Claude Code Skill，用于降低论文的 AI 文风。对 Word 文档逐章节进行风格改写，保留原有格式，每完成一个章节后等待用户确认再继续。

## 功能特点

- **逐章节改写**：按 Heading 1/2/3/4 识别章节边界，逐个处理
- **保留格式**：严格保留段落样式、字体、字号、粗体、斜体等格式信息
- **安全操作**：自动备份原文件，支持随时恢复
- **人工确认**：每章节改写完成后暂停，等待用户确认后再继续
- **精准过滤**：自动跳过标题、表格、图片等非正文段落

## 安装

将 `SKILL.md` 放置到 Claude Code 的 skill 目录中即可使用。安装后可通过 `/rewrite` 命令调用。

## 前置条件

- Python 3.x
- `python-docx` 和 `lxml` 库：

```bash
pip install python-docx lxml
```

## 使用方式

1. 在 Claude Code 中运行 `/rewrite` 命令
2. 提供需要改写的 Word 文档路径
3. 选择要从哪个章节开始改写
4. 每个章节改写完成后，选择继续、重做、跳过或退出

## 改写策略

改写后的文本风格：比原文稍微啰嗦、更具解释性、措辞更通俗，同时保持专业底线和技术准确性。

主要改写手法：
- 动词短语扩展（如"管理" → "开展...的管理工作"）
- 系统性词汇替换（如"基于" → "鉴于"）
- 括号内容整合（如"ORM（对象关系映射）" → "对象关系映射即ORM"）
- 句式微调（如"若...则..." → "如果...就..."）

## 自行编写脚本

本 Skill 不附带预置 Python 脚本，需根据 `SKILL.md` 中工作流程的要求自行编写并保存到 `scripts/` 目录下。脚本需通过 `python-docx` 操作 Word 文档，以命令行参数接收输入，输出 JSON 格式结果到 stdout。
